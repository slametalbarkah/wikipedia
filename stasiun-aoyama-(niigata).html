Stasiun Aoyama (Niigata)

Stasiun Aoyama (青山駅code: ja is deprecated , Aoyama-eki) adalah sebuah stasiun kereta api di Nishi-ku, Niigata, Prefektur Niigata, Jepang. Pengoperasian Stasiun Aoyama dilakukan oleh Perusahaan Kereta Api Jepang Timur.





== Jalur ==

Stasiun Aoyama dilayani oleh Jalur Echigo, dan berjarak 77.7 kilometer dari terminus jalur tersebut di Kashiwazaki.
