Stasiun Akagi (Nagano)

Stasiun Akagi (赤木駅code: ja is deprecated , Akagi-eki) adalah stasiun kereta api di Jalur Iida terdapat di kota Ina, Prefektur Nagano, Jepang, diopetasikan oleh Central Japan Railway Company (JR Central).





== Jalur ==

Stasiun Akagi melayani Jalur lida dan berjarak sekitar 170,4 kilometer dari titik awal jalur di Stasiun Toyohashi.





== Tata letak stasiun ==

Stasiun ini terdiri dari satu peron sisi pada permukaan tanah, dan melayani trek dua arah. Tidak ada bangunan stasiun, tetapi hanya tempat berlindung di peron. Stasiun itu tidak dijaga.





== Stasiun terdekat ==





== Sejarah ==

Stasiun Akagi dibuka pada 27 Desember 1913. Dengan privatisasi Kereta Api Nasional Jepang (JNR) pada 1 April 1987, stasiun tersebut berada di bawah kendali JR Central. Bangunan stasiun dibangun kembali pada tahun 1990.





== Statistik penumpang ==

Pada tahun fiskal 2015, stasiun ini digunakan rata-rata 60 penumpang setiap hari (hanya penumpang naik).
