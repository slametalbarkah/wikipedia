Kubernetes - Panduan Lengkap 2025

Terakhir diperbarui: 28 March 2025

Kubernetes (, biasanya ditulis sebagai K8s) adalah sistem orkestrasi kontainer sumber terbuka untuk mengotomatisasi penerapan, penskalaan, dan manajemen aplikasi komputer. Perangkat lunak ini awalnya dirancang oleh Google dan sekarang dikelola oleh Cloud Native Computing Foundation. Kubernetes bertujuan menyediakan "platform untuk mengotomatisasi penyebaran, penskalaan, dan operasi kontainer aplikasi di seluruh klaster hos". Kubernetes bekerja dengan berbagai alat kontainer dan menjalankan kontainer dalam sebuah cluster, seringkali dengan image yang dibuat menggunakan Docker. Kubernetes awalnya berinteraksi dengan runtime Docker melalui "Dockershim"; namun, shim sudah tidak digunakan lagi dan digantikan dengan antarmuka langsung menggunakan container melalui containerd, atau mengganti Docker dengan runtime yang sesuai dengan Container Runtime Interface (CRI) yang diperkenalkan oleh Kubernetes pada tahun 2016.

Mengapa Kubernetes Penting?



Banyak layanan awan yang menawarkan platform atau infrastruktur berbasis Kubernetes sebagai sebuah layanan (PaaS atau IaaS). Pada layanan ini Kubernetes dapat digunakan sebagai layanan penyedia platform. Banyak vendor juga menyediakan distribusi Kubernetes menggunakan merek mereka sendiri.
